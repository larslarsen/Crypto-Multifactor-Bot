# FX-002 - Stablecoin FX Source Feasibility Audit

**Priority:** P0
**Status:** AWAITING_REVIEW - BINANCE DIRECT-USD AUDIT COMPLETE
**Dependencies:** FX-001 readiness accepted by REVIEW-0081; AUD-003 source-audit conventions
**Layer:** research evidence / data-source feasibility
**Architecture:** no ADR, migration, or production implementation authorized

## Objective

Determine whether one repository-known official/public source can support historically
reconstructable USD-per-stablecoin observations with immutable raw evidence and defensible
point-in-time availability.

## Candidate Scope

- Kraken official stablecoin/USD market data, subject to the existing deferred bulk-source status.
- Coin Metrics Community catalog/timeseries, only if a stablecoin USD reference-rate metric is
  actually available and auditable.
- DefiLlama stablecoin/price endpoints, only to establish actual historical and availability
  behavior; do not infer it from current responses.
- Binance stablecoin cross-pairs as secondary evidence only; a stablecoin/stablecoin pair cannot
  independently anchor USD.

## Required Decision

Select exactly one auditable primary source or conclude that none is currently viable. No FX schema,
normalizer, join, migration, ADR, or downstream conversion may be implemented in this ticket.

## Stop Condition

Publish the source-feasibility report and return control to Reviewer with
`Next ticket authorized: NONE`. Do not begin implementation or another ticket.
