# MAN-001 — Dataset manifest fingerprint and publisher

**Priority:** P0  \
**Dependencies:** CAT-001, RAW-001  \
**Layer:** catalog
**Status:** IN_PROGRESS

## Objective

Publish immutable datasets whose identity is derived from canonical lineage and output hashes.

## Deliverables

- canonical manifest model and JSON schema validation;
- deterministic dataset ID/fingerprint;
- output file hash/row/byte registration;
- input raw-object and input-dataset edges;
- temporary build area and atomic final publication;
- rejection of overwrite or fingerprint/content disagreement;
- manifest verification command.

## Invariants

File paths are locations, not identities. Dataset identity includes schema, transform, code, config, inputs, and output hashes.


## Agent stop condition

After the acceptance commands pass, produce a change report and stop. Do not begin the next ticket.
