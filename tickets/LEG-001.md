# LEG-001 — Register legacy local files without accepting their claims

**Priority:** P0  \
**Dependencies:** RAW-001  \
**Layer:** ingest
**Status:** IN_PROGRESS

## Objective

Create a complete content/hash census of legacy local observations and artifacts without treating them as canonical or validated.

## Deliverables

- bounded-memory recursive scanner;
- file type, size, mtime, hash, relative path, and evidence class;
- configurable exclusions for secrets, environments, caches, and Git;
- no parsing required for registration;
- duplicates reported by content hash;
- generated inventory dataset and summary.

## Invariants

Registration does not imply acceptance. Original bytes are never rewritten. Raw data remains outside Git.


## Agent stop condition

After the acceptance commands pass, produce a change report and stop. Do not begin the next ticket.
