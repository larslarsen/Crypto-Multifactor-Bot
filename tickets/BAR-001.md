# BAR-001 — Canonical bar publisher and daily reconciliation

**Priority:** P0  
**Dependencies:** first accepted source normalizer  
**Layer:** market/quality
**Status:** IN_PROGRESS

## Objective

Publish canonical intraday bars and deterministic daily bars with explicit venue, instrument, interval, timestamp, unit, and availability semantics.

## Deliverables

- schema-conformant Parquet;
- stable sorting and uniqueness checks;
- daily resampling from accepted intraday bars;
- reconciliation against source-native daily bars where available;
- tolerance report and quarantine;
- partition sizing measured on actual data.
